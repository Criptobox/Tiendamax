💱 Tasa de cambio USD → MN

Pon la tasa del día (sin el +10 de margen, eso se suma automático). Al guardar sube a GitHub y todos los clientes la ven al instante.

El cliente verá esta tasa +10 MN de margen. Ej: si pones 340, el cliente ve 350 MN por USD.

Configura tu cuenta de GitHub para que los cambios se guarden automáticamente para todos tus clientes.

GitHub Settings → Developer Settings → Personal access tokens → Fine-grained tokens → Solo activa Repository permissions → Contents: Read and Write. Evita los tokens Classic con permiso repo completo.

⚠️ Seguridad: Este token se guarda en tu navegador (localStorage). Usa siempre un Fine-grained token con permisos mínimos (solo Contents R/W en este repositorio). Rótalo cada 90 días. Nunca compartas este dispositivo con acceso al panel admin.

🔔 Notificaciones Push (Firebase)

Configure Firebase Cloud Messaging para enviar notificaciones automáticas instantáneas al agregar nuevos productos, lanzar ofertas o cambiar la tasa de cambio.

Consigue esto en la Consola de Firebase -> Configuración del proyecto -> Tus Apps (Web App).

Consíguela en Firebase Console -> Configuración del proyecto -> Cloud Messaging -> Web configuration -> Web Push certificates.

Requerido para enviar notificaciones de prueba o manuales directamente desde este panel de control.
