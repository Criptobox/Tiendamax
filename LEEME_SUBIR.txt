⚠️  RECOMENDACIONES IMPORTANTES (no automatizadas — revisar tú)
===============================================================

1) Reglas de Firebase Realtime Database
   Tu config expone `databaseURL`. Verifica en Firebase Console que `/tokens`
   tenga reglas restrictivas, p.ej.:
   {
     "rules": {
       "tokens": {
         "$tokenId": {
           ".write": "newData.exists() && newData.hasChild('token')",
           ".read":  false
         }
       }
     }
   }
   Si está abierto, cualquiera puede leer/borrar todos los tokens.

2) Restringir la API Key de Firebase por dominio
   En Google Cloud Console → APIs & Services → Credentials → tu API key →
   "Application restrictions" → "HTTP referrers" → añadir https://tiendamax.org/*

3) Token GitHub guardado en localStorage
   Para mayor seguridad, usa GitHub fine-grained PAT solo con permiso
   "Contents: Read & Write" SOLO de este repo, y considera no marcar
   "Recordar" cuando lo pegues en el admin.

4) Las reseñas que dejan los clientes sin token quedan SOLO en su navegador
   (ahora el mensaje lo dice claramente). Si quieres que se publiquen,
   añade un buzón en el admin para revisarlas y publicarlas.
