PAQUETE SOLO PARA LA TASA AUTOMÁTICA
====================================

Este paquete contiene únicamente lo necesario para que GitHub actualice
la tasa desde elTOQUE una vez al día.

Contenido:
- config.json                                  → va en la raíz del repo
- .github/workflows/update-eltoque-rate.yml
- scripts/update_rate_from_eltoque.py

IMPORTANTE:
No pongas los archivos todos sueltos en la raíz.
Debes respetar las carpetas tal como vienen aquí.

Estructura final esperada:

repo/
├── config.json
├── scripts/
│   └── update_rate_from_eltoque.py
└── .github/
    └── workflows/
        └── update-eltoque-rate.yml

La tasa guardada en config.json es la BASE de elTOQUE.
La web sigue sumando +10 MN al mostrarla al cliente.

Notas v2:
- El script ahora valida que la tasa esté en un rango razonable (100–2000).
  Si elTOQUE devuelve algo fuera de eso, NO se actualiza (evita publicar basura).
- Si la tasa no cambia, NO se reescribe config.json (evita generar commits
  vacíos y disparar el workflow de notificaciones por nada).
